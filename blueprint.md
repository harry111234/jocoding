# Blueprint: Lotto Number Generator

## Overview

This application is a simple, user-friendly lottery number generator. It provides a clean, interactive interface for users to generate random lottery numbers. The design is modern, visually appealing, and responsive, ensuring a great experience on both desktop and mobile devices. Now includes theme support and a partnership inquiry form.

## Project Outline

### Initial Version

*   **HTML Structure (`index.html`):** Main container, title, generate button, and ball container.
*   **Styling (`style.css`):** Centered layout, vibrant colors, lottery ball styling, and responsive design.
*   **JavaScript Logic (`main.js`):** Unique number generation (1-45), UI rendering, and color assignment.

### Version 1.1: Dark Mode Support

*   **Theme Toggle:** Button to switch between Dark and Light modes.
*   **CSS Variables:** Consistent theme management.
*   **Persistence:** `localStorage` integration.

### Version 1.2: Partnership Inquiry Form

*   **Formspree Integration:** A contact form that sends submissions to a specific Formspree endpoint.
*   **Form Design:** Styled inputs and textareas that adapt to both Dark and Light modes.
